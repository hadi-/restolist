
# How to
```bash
git clone git@github.com:hadi-/restolist.git
```

```bash
cd restolist
npm install
npm run build
npm run start-dev
```

Production scripts
it will build & run in production mode 

```bash
npm run build:production
npm start
```

it should be run on localhost:3000
